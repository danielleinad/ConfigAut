
# ConfigAut
Repositorio de prueba para Mis configuraciones automaticas en Linux

utiliza el archivo backup.sh para guardar las configuraciones. 
estas se guardan en el archibo Dotfiles. 

El archibo Dotfiles son las configuraciones personalizadas que tienes 
despues de ejecutar el archivo backup. 

el archivo install.sh utilizara la carpeta dotfiles de configuracion 
para automatizar las configuraciones de los paquetes que insalara
